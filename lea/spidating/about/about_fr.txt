Logiciel SpiDating : Spirea-Dating

Ce logiciel est un programme informatique servant à la création en masse d'évènements de calendrier. Il permet aussi de modifier en masse les statuts des évènements et de voir les statistiques de temps par éléments (projets, infolog, ticket spid, etc...)

Etapes d'installation
- copier le répertoire spidating dans la racine Egroupware, au même niveau que les autres modules
- aller dans le setup egroupware, installer l'application spidating
- dans egroupware, depuis un compte administrateur : 
-- Menu Admin/Applications : vérifier que l'application est installée
-- Menu Admin/Groupes : donner accès à spidating

Etapes de configuration
- les administrateurs egroupware ont accès au menu "Admin" de l'application 

1. Admin > configuration du site
- Général : déterminez votre configuration
